Az első webalkalmazásom

Ezen a héten egy kisebb webes alkalmazáson fogsz dolgozni ami egy képgalériához hasonlóan fog működni, vagyis képeket fog megjeleníteni a címükkel és leírásukkal. Lejjebb találsz egy részletes leírást ami elmondja hogyan kell működnie az alkalmazásnak kész állapotában. A bonyolultabb funkciókat kihagyhatod ha gondolod és majd később visszatérhetsz ezekhez. Illetve csináltunk neked egy napokra lebontott tervet is ami segíthet abban, hogy hogyan érdemes haladni. Viszont ez csak egy javaslat, ha úgy érzed akkor nyugodtan térj el tőle.

Látványterv

Specifikáció

Mielőtt még elkezded

  • Fejben tervezd meg mit is akarsz csinálni
  • Készíts sketch-upokat amik majd segítenek a HTML és CSS megírásában
  • Tervezd meg az adatstruktúrát(data structure) amiben tárolni fogod a szükséges adatokat JavaScript oldalon
  • Gondolkodj egy kicsit, hogy mik azok amiket több helyen is fel tudsz használni és tervezz nekik függvényeket és változókat
  • Gondold át mik lesznek a user eventek amiket le kell kezelned

Vázlat és működés

  • Nyugodtan változtass a designon ha szeretnél, de fontos, hogy jól nézzen ki
  • A szerkezet a képernyő közepén helyezkedjen el és fix mérete legyen
  • A galéria két fő részből áll. Az egyik ahol nagyban látszik egy kép, illetve alul a thumbnailek(indexképek)
  • A felső rész tartalmaz két gombot nyilakkal az oldalakon, illetve középen az aktuálisan kiválasztott képet.
  • Alul jelenjenek meg a thumbnailek, vagyis az összes kép kisméretben
  • Az aktuálisan kiválasztott kép alján egy félig átlátszó blokk helyezkedik el ami tartalmazza a kép címét és leírását

Navigáció és interakció

  • Mikor először megnyitjuk az oldalt, az eltárolt képek közül az első jelenik meg nagyban
  • Az ehhez tartozó thumbnail(ami az első) pedig aktív(outline, border, stb…)
  • Mikor rákattintunk a nyilakra, a középen lévő nagy kép(és az ehhez tartozó cím és leírás) átvált a következőre vagy az előzőre a gombtól függően
  • Ugyanekkor az éppen aktív thumbnail is változik az aktuálisan kiválasztott kép alapján
  • Ha épp a legelső kép van kiválasztva, akkor a visszafele nyílra kattintáskor az utolsó kerül kiválasztásra
  • Hasonlóan, ha az utolsó kép az aktív, akkor az előre nyílra kattintáskor az első kép kerül kiválasztásra
  • Bármelyik thumbnailre kattintáskor az adott kép kerül kiválasztásra
  • A nyilak hatása, az éppen megjelenített nagy kép és az aktív thumbnail mindig legyenek egymással szinkronban

Terv napokra lebontva

Első nap

  • Készíts egy folder a projectednek
  • Hozd létre a HTML, CSS és JavaScript fájlokat és kösd össze őket egymással
  • Töltsd be a jQuery-t is a HTML-ben
  • Készíts egy images foldert és helyezz bele néhány képet amiket majd megjelenítesz a galériádban
  • Készítsd el azt a részt a HTML-ben ahol nagyban jelenik meg kép egyelőre egy darab képpel, majd csináld meg a CSS részt is hozzá(A thumbnailekkel egyelőre még ne foglalkozz, az később jön)
  • A nyilakhoz itt találsz egy képet amit letölthetsz és használhatsz
    • Ezt meg lehet tükrözni CSS segítségével a másik oldalhoz

Második nap

  • Szedd ki a képet(az src attribútumot az img tagből) és a szöveget a cím és leírás HTML tagekből és adj a tagekhez id-kat ha eddig még nem tetted meg
<img id="photo">
<div class="gray-container">
  <h1 id="photo-title"></h1>
  <p id="photo-description"></p>
</div>
  • Tedd őket egy JavaScript objectbe aminek 3 fieldje van (photo, title, desciption)
  • jQuery segítségével töltsd be a képet és a szöveget a megfelelő helyre a HTML-ben
let data = {
  photo: 'images/photo1.jpg',
  title: 'My title',
  description: 'What happened here, why is this a very nice image'
};

$('#photo').attr('src', data.photo);
// ...
  • Készíts egy arrayt amibe tegyél egy a fentihez hasonló objectet minden képhez ami az images folderben található
  • Hozz létre egy változót (pl. currentPhoto) amiben eltárolod az aktuálisan megjelenített kép indexét (ez kezdetben lehet 0)
  • Amikor szükséged van adatra az arrayből, használd a currentPhoto változódat indexként
let currentPhoto = 0;
let imagesData = [...];
$('#photo').attr('src', imagesData[currentPhoto].photo);
// ...

Harmadik nap

  • Azt a kódot ami a jQuerz segítségével betölt egy képet, tedd egy olyan függvénybe ahol a kép indexe az input
  • Ezt a függvényt 0-val hívd meg az oldal betöltésekor, majd a nyilakra kattintáskor változtasd a currentPhoto változó értékét és ezt a változót add meg a függvénynek inputként
let loadPhoto = (photoNumber) => {
  $('#photo').attr('src', imagesData[photoNumber].photo);
  // ...
}

loadPhoto(currentPhoto);

$('#right-arrow').click(() => {
  currentPhoto++;
  loadPhoto(currentPhoto);
})
  • Biztosítsd, hogy a currentPhoto változó értéke nem lehet nagyobb mint az array mérete amiben az adatok vannak, illetve nem lehet kisebb mint 0
  • Kis segítség

Negyedik nap

  • Készítsd el a keretet amiben a thumbnailek lesznek
  • Tegyél bele egy-két thumbnailt HTML-ben és formázd meg őket CSS segítségével
  • Egyelőre csak a thumbnailekkel foglalkozz, a szöveggel ami akkor jelenik meg mikor a kurzor felette van ráérsz később
  • Hozd létre a thumbnaileket JavaScriptben. Használd a forEach functiont az arrayen amiben a képeket tárolod
  • Implementálj egy függvényt ami lekezeli a thumbnailekre történő kattintásokat
  • Tudnod kell, hogy melyik thumbnailre történt a kattintás
    • A legegyszerűbb ha ezt az információt a HTML tagekbe helyezed amikor létrehozod őket
    • Általánosságban bármilyen egyedi információt el tudsz tárolni a tagekben ha a data- kezdetű attribútumokat használod. Pl: data-number="1"
    • Ezt később el tudod érni a következőképpen: $(event.target).attr('data-number')
    • Kis segítség

Ötödik nap

  • Látszódjon, hogy melyik thumbnail aktív (egészítsd ki a loadPhoto függvényedet)
  • Készítsd el a szöveget ami megjelenik amikor a thumbnail felett van a kurzor
  • Ezt például megtehet úgy, hogy létrehozod és pozícionálod a szükséges HTML taget, de alapból rejtve van és csak akkor látszik ha a kurzor az adott thumbnail felett van
    • Kis segítség
  • Add anything else you’d like :)
